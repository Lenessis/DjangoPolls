# DjangoPolls
Należy utworzyć nowy projekt Django w PyCharm Pro i wkleić tam te pliki.
W terminalu Pycharma wpisać:
  py manage.py runserver //(windows)
  python manage.py runserver //(linux)
Otworzyć przeglądarkę i wpisać localhost:8000/polls
Aby zalogować się na localhost:8000/admin
  login: admin
  haslo: 123
